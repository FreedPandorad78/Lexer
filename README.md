# Lexer de Java

Analizador léxico (lexer) para un subconjunto del lenguaje Java.
Hecho como proyecto de clase para el curso de Compiladores.

## ¿Qué hace?

Lee código fuente Java y lo descompone en tokens: palabras clave, identificadores,
operadores, literales, etc. Muestra la línea y columna donde aparece cada token.

## ¿Cómo compilar?

```bash
javac *.java
